Get all ad hoc mailings.

GET /ad_hoc_mailings

Tags: adHocMailings

Get all ad hoc mailings.

'Per', 'page', 'query' and 'state' param allowed.

page: integer 1

Page number

per: integer 10

Number of ad hoc mailings returned per page

query: string

Filters ad hoc mailings by name or title.

state: string

When not specified all objects are return regardless their state. Allowed states:

  • enabled
  • disabled
  • archived
  • not_archived

Uses default content-types: application/json

200 OK

Operation successful

Create new ad hoc mailing.

POST /ad_hoc_mailings

Tags: adHocMailings

Create new ad hoc mailing.

ad_hoc_mailing: object
title: string

Ad hoc mailing's new title. Name will defined from this attribute e.g.:

  • 'New Ad hoc mailing' will have name 'new_ad_hoc_mailing'
  • 'NewAdHocMailing' will have name 'newadhocmailing'
subject: string

Ad hoc mailing's subject.

template: string

Ad hoc mailing's template. Will be validated against Liquid syntax.

list: string

Previously created MailyHerald::List object's name.

from: string

If not specified, default value will be used.

conditions: string

After validation has to return boolean value.

mailer_name: string

Has to be previously created mailer.
